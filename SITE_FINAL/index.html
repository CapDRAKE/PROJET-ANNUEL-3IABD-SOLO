Le développeur


Bastien LEUWERS

Capdrake

Comprendre la reflexion humaine pour reconnaître une image afin de mieux appliquer notre modèle

L'oeil humain sait reconnaître une image même si celle-ci est de mauvaise qualité. Il va se fier à différents paramètres afin de reconnaître l'image

L'objectif était alors de comprendre le fonctionnement de l'oeil humain afin de l'appliquer au mieux à notre analyse !

Un dataset, 3 classes

Un dataset composé de 3 classes divisées en 2 catégories : train et test

Un seul modèle, le MLP

Dans le cadre de ce projet, je devais développer un Perceptron multi-couche

Un application

A la fin, il fallait réaliser une application permettant de prédire une image

Le développement

MLP en python

L'ensemble du MLP a été réalisé en python. Il a fallu plusieurs versions de ce MLP afin d'obtenir de bon résultats

Les librairies

J'ai utilisé plusieurs librairies afin de réaliser mon MLP tel que Numpy pour les calculs mais aussi matplotlib

Constitution du dataset

Pour constituer mon dataset, j'ai utilisé des images d'internet, mais également de kaggle etc...

Méthode de train des images

Pour entrainer mon modèle avec mes images sur 3 classes, j'ai utilisé la méthode OneHotEncoder

Création d'une API pour l'application

Pour faire fonctionner notre modèle, j'ai créé une API permettant d'upload l'image et de prédire

Site web et application flask

Dans un premier temps, une toute petite application flask a été réalisé. L'objectif aurait été ensuite de l'implémenter sur le site
